# Keychron Q3 HE — Hardware Design Files

> TKL magnetic switch custom keyboard with a programmable knob, adjustable actuation, rapid trigger, and tri-mode connectivity.

**Product page:** [Keychron Q3 HE](https://www.keychron.com/products/keychron-q3-he-qmk-wireless-custom-keyboard)

![Q3 HE](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron-Q3-HE-page1.jpg)

## Specifications

| Spec | Detail |
|---|---|
| Layout | TKL / 80% (87-key) |
| Mount | Double-gasket |
| Case Material | Full aluminum |
| Knob | Yes |
| Switch Type | Gateron Double-Rail Magnetic Nebula Switch |
| Connectivity | 2.4 GHz / Bluetooth 5.2 / USB-C |
| Polling Rate | 1000 Hz (wired and 2.4 GHz) / 90 Hz (Bluetooth) |
| Rapid Trigger | Yes |
| Adjustable Actuation | Yes |
| Firmware | Keychron Launcher / QMK |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/hall_effect_playground/keyboards/keychron/q3_he) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.stp` | FreeCAD, Onshape | Fusion 360, SolidWorks |
| `.dxf` | LibreCAD, QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |
