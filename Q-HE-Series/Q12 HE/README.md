# Keychron Q12 HE — Hardware Design Files

> 96% Hall Effect keyboard files for the Q12 HE, including enclosure parts, knob, and a complete assembled model.

**Product page:** [Keychron Q12 HE](https://www.keychron.com/products/keychron-q12-he-qmk-wireless-custom-keyboard)

![Q12 HE](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron_Q12_HE_Wireless_QMK_Custom_Magnetic_Switch_Keyboard-3.jpg?v=1754620571)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 96% |
| Knob | Yes |
| Switch Type | Magnetic / Hall Effect |
| Connectivity | 2.4 GHz / Bluetooth / USB-C |
| Firmware | Keychron Launcher / QMK |

## Included Files

| File | Description |
|---|---|
| `Q12-HE-Top-Case.stp` | Top case enclosure |
| `Q12-HE-Bottom-Case.stp` | Bottom case enclosure |
| `Q12-HE-Knob.stp` | Rotary encoder knob model |
| `Q12-HE-US-Aluminu-Plate.dwg` | US aluminum plate drawing |
| `Q12-HE-US-Aluminu-Plate.pdf` | US aluminum plate reference sheet |
| `Q12-HE-US-Full-Model-20250801.stp` | Complete assembled US model |
| `Stabilizer-2.25u.stp` | 2.25u stabilizer model |
| `Stabilizer-6.25u.stp` | 6.25u stabilizer model |
