# Keychron Q5 Pro — Hardware Design Files

> 96% wireless custom mechanical keyboard with a programmable knob, Bluetooth 5.1, and USB-C wired connectivity.

**Product page:** [Keychron Q5 Pro](https://www.keychron.com/products/keychron-q5-pro-qmk-via-wireless-custom-mechanical-keyboard)

![Q5 Pro](https://i.imgur.com/3z9mMMI.jpg)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 96% (97-key) |
| Mount | Double-gasket |
| Case Material | Full aluminum |
| Knob | Yes |
| Connectivity | Bluetooth 5.1 / USB-C |
| Polling Rate | 1000 Hz (wired) / 90 Hz (wireless) |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/bluetooth_playground/keyboards/keychron/q5_pro) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.step` | FreeCAD, Onshape | Fusion 360, SolidWorks |
| `.dxf` | LibreCAD, QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |
