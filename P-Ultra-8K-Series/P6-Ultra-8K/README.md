# Keychron P6 Ultra 8K — Hardware Design Files

> 100% full-sized wireless custom mechanical keyboard with a full-metal body, 8K polling, ZMK firmware, and a rapid-assembly screwless structure.

**Product page:** [Keychron P6 Ultra 8K](https://www.keychron.com/products/keychron-p6-ultra-8k-wireless-custom-mechanical-keyboard)

![P6 Ultra 8K](https://www.keychron.com/cdn/shop/files/P6-Ultra-Iconic-Features.jpg?crop=center&height=1200&v=1767005341&width=1200)

## Current Repository Status

This folder already includes the following hardware design assets for the ANSI / US version:

| File | Description | Format |
|---|---|---|
| `P6-Ultra-8K-US-Cherry-Profile-Keycap-Full-Model-20260130.stp` | Full 3D model with Cherry-profile keycaps | STEP |
| `P6-Ultra-8K-US-Plate-20260413.dwg` | Plate drawing for CAD workflows | DWG |
| `P6-Ultra-8K-US-Plate-20260413.dxf` | Plate drawing for laser/CAD compatibility workflows | DXF |
| `P6-Ultra-8K-US-Plate-20260413.pdf` | Plate reference drawing for quick viewing | PDF |

## Product Snapshot

| Spec | Detail |
|---|---|
| Layout | 100% / full size |
| Knob | Yes |
| Case Material | Full metal |
| Connectivity | 2.4 GHz / Bluetooth 5.3 / USB-C wired |
| Polling Rate | Up to 8,000 Hz |
| Firmware | ZMK |
| Mount Style | Gasket |
| Keycaps | Cherry profile double-shot PBT |
| Switch Support | MX-style hot-swappable |

## Typical File Uses

| File Type | Typical Use |
|---|---|
| `stp` | Full assembly, case, plate, or accessory 3D CAD work |
| `dwg` / `dxf` | Plate geometry, manufacturing references, and accessory fitting |
| `pdf` | Quick-view drawings and dimension reference |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.stp` | FreeCAD, Onshape | Fusion 360, SolidWorks |
| `.dwg` / `.dxf` | LibreCAD, QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |

## Notes

- The currently uploaded files are for the US / ANSI plate and full-model variant.
- File names include date stamps to make revision tracking easier as additional variants are added.
- If more case, PCB, or regional-layout files are released later, they can be added alongside the current assets in this folder.
