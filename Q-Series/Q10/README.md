# Keychron Q10 — Hardware Design Files

> 75% Alice-layout custom mechanical keyboard with extra macro support and ergonomic positioning.

**Product page:** [Keychron Q10](https://www.keychron.com/products/keychron-q10-alice-layout-qmk-custom-mechanical-keyboard)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% Alice |
| Size | 82-key |
| Case Material | CNC aluminum |
| Connectivity | USB-C wired |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/master/keyboards/keychron/q10) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications, where applicable |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.step` | FreeCAD, Onshape | Fusion 360, SolidWorks |
| `.dxf` | LibreCAD, QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |
