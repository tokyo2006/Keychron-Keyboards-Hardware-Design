# Keychron Q4 — Hardware Design Files

> 60% custom mechanical keyboard with a compact layout and full aluminum construction.

**Product page:** [Keychron Q4](https://www.keychron.com/products/keychron-q4-qmk-via-custom-mechanical-keyboard)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 60% |
| Size | 61-key |
| Case Material | CNC aluminum |
| Connectivity | USB-C wired |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/master/keyboards/keychron/q4) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications, where applicable |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.step` | FreeCAD, Onshape | Fusion 360, SolidWorks |
| `.dxf` | LibreCAD, QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |
