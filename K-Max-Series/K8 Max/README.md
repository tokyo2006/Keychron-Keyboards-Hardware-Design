# Keychron K8 Max — Hardware Design Files

> TKL 87-key wireless mechanical keyboard with tri-mode connectivity and 1000 Hz polling rate.

**Product page:** [Keychron K8 Max](https://www.keychron.com/products/keychron-k8-max-qmk-wireless-mechanical-keyboard)

![K8 Max](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron_K8_Max_Wireless_Mechanical_Keyboard.jpg?v=1722679778)

## Specifications

| Spec | Detail |
|---|---|
| Layout | TKL / 80% (87-key) |
| Case Material | ABS / ABS + Aluminum |
| Profile | Standard |
| Knob | No |
| Connectivity | Bluetooth 5.1 / 2.4 GHz / USB-C |
| Polling Rate | 1000 Hz (2.4 GHz) |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/wireless_playground/keyboards/keychron/k8_max) |

## Included Files

| File | Description |
|---|---|
| `bottom-case` | 3D CAD model of the bottom enclosure (top case not yet available) |
| `plate` | Switch plate with ANSI cutout dimensions |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
