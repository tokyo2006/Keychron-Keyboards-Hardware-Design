# Keychron M3 Mini — Hardware Design Files

> Compact lightweight wireless optical mouse with ergonomic shaping, tri-mode connectivity, and high-performance sensor options.

**Product page:** [Keychron M3 Mini Wireless Mouse](https://www.keychron.com/products/keychron-m3-mini-wireless-mouse)

![Keychron M3 Mini](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron-M3-Mini-White-13.jpg?v=1686381609)

## Specifications

| Spec | Detail |
|---|---|
| Type | Wireless optical mouse |
| Shape | Ergonomic right-handed compact |
| Weight | 55 g |
| Sensor | PixArt 3395 / PixArt 3950 (version-dependent) |
| DPI | Up to 30000 |
| Polling Rate | Up to 4000 Hz or 8000 Hz (version-dependent), 125 Hz (Bluetooth) |
| Connectivity | Bluetooth / 2.4 GHz / USB-C wired |
| Switch | Huano 80M micro switch |
| Battery | 600 mAh |
| Body Material | ABS |
