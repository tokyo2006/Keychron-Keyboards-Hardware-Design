# Keychron M5 — Hardware Design Files

> Wireless ergonomic mouse with vertical shaping, thumb wheel, tri-mode connectivity, and up to 8K polling.

**Product page:** [Keychron M5 Wireless Ergonomic Mouse](https://www.keychron.com/products/keychron-m5-wireless-ergonomic-mouse)

![Keychron M5](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/M5-8K-2.jpg?v=1743646928)

## Specifications

| Spec | Detail |
|---|---|
| Type | Wireless ergonomic mouse |
| Shape | Vertical ergonomic right-handed |
| Weight | 95 +/- 3 g |
| Sensor | PixArt 3950 |
| DPI | 100 - 30000 |
| Polling Rate | Up to 8000 Hz (2.4 GHz / wired), 125 Hz (Bluetooth) |
| Connectivity | Bluetooth 5.3 / 2.4 GHz / USB-C wired |
| Switch | Huano micro switch |
| Battery | 600 mAh |
| Body Material | ABS |
