# Keychron M7 — Hardware Design Files

> Wireless optical mouse with ergonomic shaping, tri-mode connectivity, and high-performance sensor options for work and play.

**Product page:** [Keychron M7 Wireless Mouse](https://www.keychron.com/products/keychron-m7-wireless-mouse)

![Keychron M7](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/M7_8K_4.jpg?v=1741230333)

## Specifications

| Spec | Detail |
|---|---|
| Type | Wireless optical mouse |
| Shape | Ergonomic right-handed |
| Weight | Lightweight ergonomic form factor |
| Sensor | PixArt 3395 / PixArt 3950 (version-dependent) |
| DPI | Up to 30000 |
| Polling Rate | Up to 4000 Hz or 8000 Hz (version-dependent), 125 Hz (Bluetooth) |
| Connectivity | Bluetooth / 2.4 GHz / USB-C wired |
| Switch | Huano micro switch |
| Battery | 600 mAh |
| Body Material | ABS |
