# Keychron M6 — Hardware Design Files

> Wireless productivity mouse with ergonomic shaping, thumb wheel, tri-mode connectivity, and high polling-rate support.

**Product page:** [Keychron M6 Wireless Mouse](https://www.keychron.com/products/keychron-m6-wireless-mouse)

![Keychron M6](https://www.keychron.com/cdn/shop/files/M6-Mouse-Hero-Image-on-Desktop.jpg?v=1769594628&width=3200)

## Specifications

| Spec | Detail |
|---|---|
| Type | Wireless optical mouse |
| Shape | Ergonomic right-handed |
| Weight | Ultra-light ergonomic form factor |
| Sensor | PixArt 3395 / PixArt 3950 (version-dependent) |
| DPI | Up to 30000 |
| Polling Rate | Up to 4000 Hz or 8000 Hz (version-dependent), 125 Hz (Bluetooth) |
| Connectivity | Bluetooth / 2.4 GHz / USB-C wired |
| Switch | Huano micro switch |
| Battery | 800 mAh |
| Body Material | ABS |
