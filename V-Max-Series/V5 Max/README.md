# Keychron V5 Max — Hardware Design Files

> 96% layout, 97-key wireless gasket-mounted custom mechanical keyboard with rotary encoder knob. Tri-mode connectivity (Bluetooth 5.1 / 2.4 GHz / USB-C).

**Product page:** [Keychron V5 Max](https://www.keychron.com/products/keychron-v5-max-qmk-via-wireless-custom-mechanical-keyboard)

![V5 Max](https://cdn.shopify.com/s/files/1/0059/0630/1017/files/Keychron-V5-Max-QMK-VIA-wireless-mechanical-keyboard.jpg?v=1702977391)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 96% (97-key) |
| Case Material | Plastic |
| Mount | Gasket |
| Knob | Yes (rotary encoder) |
| Connectivity | Bluetooth 5.1 / 2.4 GHz / USB-C |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/wireless_playground/keyboards/keychron/v5_max) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
