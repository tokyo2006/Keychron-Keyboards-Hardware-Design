# Keychron V8 Max — Hardware Design Files

> Alice layout, 69-key wireless gasket-mounted custom mechanical keyboard with rotary encoder knob. Tri-mode connectivity (Bluetooth 5.1 / 2.4 GHz / USB-C).

**Product page:** [Keychron V8 Max](https://www.keychron.com/products/keychron-v8-max-alice-layout-qmk-custom-mechanical-keyboard)

## Specifications

| Spec | Detail |
|---|---|
| Layout | Alice (69-key) |
| Case Material | Plastic |
| Mount | Gasket |
| Knob | Yes (rotary encoder) |
| Connectivity | Bluetooth 5.1 / 2.4 GHz / USB-C |
| Hot-swap | Yes |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/wireless_playground/keyboards/keychron/v8_max) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with Alice layout cutout dimensions |
| `encoder-knob` | Rotary encoder housing and knob specifications |
| `us-full-model` | Complete assembled model (case + plate + keycaps) |
| `stabilizer` | Stabilizer housing and wire specifications |
