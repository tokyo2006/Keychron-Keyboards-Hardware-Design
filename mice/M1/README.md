# Keychron M1 — Hardware Design Files

> Lightweight wireless optical mouse with ergonomic shaping, tri-mode connectivity, and a PixArt PAW 3395 sensor.

**Product page:** [Keychron M1 Wireless Mouse](https://www.keychron.com/collections/keychron-m-series-mice/products/keychron-m1-wireless-mouse)

## Specifications

| Spec | Detail |
|---|---|
| Type | Wireless optical mouse |
| Shape | Ergonomic right-handed |
| Weight | 79 g |
| Sensor | PixArt PAW 3395 |
| DPI | 100 - 26000 |
| Polling Rate | 1000 Hz (2.4 GHz / wired), 125 Hz (Bluetooth) |
| Connectivity | Bluetooth 5.1 / 2.4 GHz / USB-C wired |
| Switch | Kailh GM 8.0 micro switch |
| Battery | 600 mAh |
| Body Material | ABS |

## Included Files

| File | Description |
|---|---|
| `shell-body` | 3D CAD model of the upper and lower mouse shell |
| `side-buttons` | Side-button geometry and fit references |
| `full-model` | Complete assembled mouse model |

## Opening the Files

| Format | Free Software | Commercial Software |
