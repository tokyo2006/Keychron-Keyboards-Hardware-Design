# Keychron K2 Pro — Hardware Design Files

> 75% wireless mechanical keyboard with QMK/VIA support, Bluetooth 5.1, and USB-C wired connectivity.

**Product page:** [Keychron K2 Pro](https://www.keychron.com/products/keychron-k2-pro-qmk-via-wireless-mechanical-keyboard)

## Specifications

| Spec | Detail |
|---|---|
| Layout | 75% (84-key) |
| Profile | Standard |
| Case Material | ABS / ABS + Aluminum |
| Knob | No |
| Connectivity | Bluetooth 5.1 / USB-C |
| Polling Rate | 1000 Hz (wired) |
| Hot-swap | Optional (model-dependent) |
| Firmware | QMK / VIA |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
