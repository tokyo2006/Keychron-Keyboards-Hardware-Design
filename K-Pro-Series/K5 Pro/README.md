# Keychron K5 Pro — Hardware Design Files

> Full-size low-profile wireless mechanical keyboard with QMK/VIA support, Bluetooth 5.1, and USB-C wired connectivity.

**Product page:** [Keychron K5 Pro](https://www.keychron.com/products/keychron-k5-pro-qmk-via-wireless-custom-mechanical-keyboard)

## Specifications

| Spec | Detail |
|---|---|
| Layout | Full-size (108-key) |
| Profile | Low-profile |
| Case Material | ABS + Aluminum |
| Knob | No |
| Connectivity | Bluetooth 5.1 / USB-C |
| Polling Rate | 1000 Hz (wired) |
| Hot-swap | Optional (RGB hot-swappable version) |
| Firmware | QMK / VIA |
| Firmware Source | [Link](https://github.com/Keychron/qmk_firmware/tree/bluetooth_playground/keyboards/keychron/k5_pro) |

## Included Files

| File | Description |
|---|---|
| `top-case-and-bottom-case` | 3D CAD model of the top and bottom enclosure |
| `plate` | Switch plate with ANSI cutout dimensions |
| `us-full-model` | Complete assembled model (case + plate + keycaps, ANSI) |
| `stabilizer` | Stabilizer housing and wire specifications |

## Opening the Files

| Format | Free Software | Commercial Software |
|---|---|---|
| `.step` | FreeCAD, Onshape | Fusion 360, SolidWorks |
| `.dxf` | LibreCAD, QCAD | AutoCAD, DraftSight |
| `.pdf` | Any PDF viewer | — |
